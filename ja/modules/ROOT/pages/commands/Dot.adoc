= Dot コマンド
:page-en: commands/Dotifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Dot( <ベクトル>, <ベクトル> )::
  2つのベクトルの内積（スカラー積）を返す．

[EXAMPLE]
====

`++Dot({1, 3, 2}, {0, 3, -2})++` 出力： _5_, _{1, 3, 2}_ と __{0, 3, -2}__のスカラー積．

====

[NOTE]
====

参照： xref:/commands/Cross.adoc[Cross コマンド].

====
